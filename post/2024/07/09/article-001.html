วิธีติดตั้ง Windows 10,11 ผ่าน Flash Drive อย่างละเอียด

ขั้นตอนที่ 1: เตรียมอุปกรณ์

  1. Flash Drive: ความจุอย่างน้อย 8GB และไม่มีข้อมูลสำคัญ (จะถูกฟอร์แมต)
  2. คอมพิวเตอร์: ที่มีการเชื่อมต่ออินเทอร์เน็ต

ขั้นตอนที่ 2: ดาวน์โหลด Windows 11

  1. ไปที่ เว็บไซต์ดาวน์โหลด Windows 11
  2. คลิกที่ "Download now" ใต้หัวข้อ Create Windows 11 Installation Media เพื่อดาวน์โหลด Media Creation Tool
  3. เมื่อดาวน์โหลดเสร็จ เปิด Media Creation Tool

ขั้นตอนที่ 3: สร้าง Flash Drive สำหรับติดตั้ง Windows 11

  1. เปิด Media Creation Tool ที่ดาวน์โหลดมา
  2. ยอมรับข้อตกลงการใช้งาน
  3. เลือก Create installation media (USB flash drive, DVD, or ISO file) for another PC แล้วคลิก Next
  4. เลือกภาษาและรุ่นของ Windows 11 ที่ต้องการติดตั้ง จากนั้นคลิก Next
  5. เลือก USB flash drive แล้วคลิก Next
  6. เลือก Flash Drive ที่ต้องการใช้ แล้วคลิก Next
  7. รอจนกว่าเครื่องมือจะดาวน์โหลดและสร้าง Flash Drive ติดตั้ง Windows 11 เสร็จ

ขั้นตอนที่ 4: ติดตั้ง Windows 11 จาก Flash Drive

  1. เสียบ Flash Drive ที่สร้างไว้กับคอมพิวเตอร์ที่ต้องการติดตั้ง Windows 11
  2. รีสตาร์ทคอมพิวเตอร์ แล้วเข้าสู่ BIOS/UEFI (กด F2, F12, Delete หรือ Esc ตามแต่ละยี่ห้อของคอมพิวเตอร์)
  3. ตั้งค่าให้บูทจาก Flash Drive ก่อน (Boot Priority) แล้วบันทึกและออกจาก BIOS/UEFI
  4. เมื่อคอมพิวเตอร์บูทจาก Flash Drive จะเห็นหน้าจอการติดตั้ง Windows 11
  5. เลือกภาษาที่ต้องการ แล้วคลิก Next
  6. คลิกที่ Install now
  7. ใส่ Product Key ถ้ามี (ถ้าไม่มีสามารถคลิก I don't have a product key ได้)
  8. เลือกรุ่นของ Windows 11 ที่ต้องการติดตั้ง แล้วคลิก Next
  9. ยอมรับข้อตกลงการใช้งาน แล้วคลิก Next
  10. เลือก Custom: Install Windows only (advanced)
  11. เลือกดิสก์ที่ต้องการติดตั้ง Windows 11 แล้วคลิก Next
  12. รอจนกระทั่งการติดตั้งเสร็จสมบูรณ์และคอมพิวเตอร์รีสตาร์ท

ขั้นตอนที่ 5: ตั้งค่าหลังการติดตั้ง

  1. เมื่อคอมพิวเตอร์รีสตาร์ทเข้าสู่ Windows 11 ครั้งแรก จะมีการตั้งค่าเริ่มต้นเช่น การตั้งค่าภาษา เขตเวลา บัญชีผู้ใช้ ฯลฯ
  2. ทำตามคำแนะนำบนหน้าจอเพื่อเสร็จสิ้นการตั้งค่าเริ่มต้น

หลังจากตั้งค่าเสร็จสิ้น คอมพิวเตอร์ของคุณจะพร้อมใช้งาน Windows 11 เรียบร้อยแล้วครับ!
