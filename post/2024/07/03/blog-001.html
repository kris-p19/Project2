วิธีที่ 1: ใช้ Command Prompt (CMD)

  1. เสียบ USB ที่ติดไวรัสเข้ากับคอมพิวเตอร์ของคุณ
  2. เปิด Command Prompt (CMD) ด้วยสิทธิ์ผู้ดูแลระบบ (Admin) โดยการคลิกขวาที่ Start แล้วเลือก Command Prompt (Admin) หรือ PowerShell (Admin)
  3. ใน Command Prompt พิมพ์คำสั่งต่อไปนี้แล้วกด Enter: attrib -h -r -s /s /d X:\*.*
    หมายเหตุ: แทนที่ X ด้วยตัวอักษรของ USB ที่ติดไวรัส (เช่น D: หรือ E:)

วิธีที่ 2: ใช้โปรแกรมป้องกันไวรัส

  1. ดาวน์โหลดและติดตั้งโปรแกรมป้องกันไวรัสที่เชื่อถือได้ เช่น Avast, Bitdefender, Kaspersky หรือโปรแกรมป้องกันไวรัสอื่นๆ ที่คุณมี
  2. เสียบ USB ที่ติดไวรัสเข้ากับคอมพิวเตอร์ของคุณ
  3. เปิดโปรแกรมป้องกันไวรัสและสแกน USB เพื่อตรวจหาและลบไวรัส
  4. หลังจากที่โปรแกรมป้องกันไวรัสลบไวรัสเสร็จเรียบร้อยแล้ว คุณสามารถใช้ USB ได้ตามปกติ

วิธีที่ 3: ใช้ File Explorer

  1. เสียบ USB ที่ติดไวรัสเข้ากับคอมพิวเตอร์ของคุณ
  2. เปิด File Explorer (Windows Explorer) แล้วไปที่เมนู View
  3. คลิกที่ Options จากนั้นเลือก Change folder and search options
  4. ไปที่แท็บ View และทำเครื่องหมายถูกที่ "Show hidden files, folders, and drives" และยกเลิกการทำเครื่องหมายที่ "Hide protected operating system files (Recommended)" แล้วคลิก OK
  5. กลับไปที่ USB ของคุณ คุณจะเห็นไฟล์และโฟลเดอร์ที่ถูกซ่อนไว้ สามารถลบไฟล์ที่น่าสงสัยได้

วิธีที่ 4: Format USB

หากวิธีข้างต้นไม่สามารถแก้ไขปัญหาได้ คุณสามารถ Format USB ได้ตามขั้นตอนนี้

  1. เสียบ USB ที่ติดไวรัสเข้ากับคอมพิวเตอร์ของคุณ
  2. เปิด File Explorer (Windows Explorer)
  3. คลิกขวาที่ USB Drive แล้วเลือก Format
  4. เลือก File System เป็น FAT32 หรือ NTFS แล้วคลิก Start
  5. รอจนกระทั่งการ Format เสร็จสิ้น

คำแนะนำเพิ่มเติม

  • ควรสำรองข้อมูลที่สำคัญจาก USB ก่อนทำการแก้ไขใดๆ
  • อย่าเสียบ USB ที่ติดไวรัสเข้ากับอุปกรณ์อื่นๆ เพื่อป้องกันการแพร่กระจายของไวรัส
  • ใช้โปรแกรมป้องกันไวรัสและมัลแวร์เพื่อป้องกันไม่ให้ติดไวรัสในอนาคต
