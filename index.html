แก้ไขปัญหาไวรัสที่ซ่อนไฟล์ใน USB คุณสามารถทำตามขั้นตอนเหล่านี้:

  • 1. สแกนไวรัส: ใช้โปรแกรมตรวจจับไวรัสเพื่อสแกน USB drive เพื่อตรวจสอบว่ามีไวรัสหรือไม่ โปรแกรมตรวจจับไวรัสที่แนะนำรวมถึง Avast, AVG, Malwarebytes, และอื่นๆ
  • 2. แสดงไฟล์ที่ถูกซ่อน: เปิด Command Prompt (หรือ PowerShell) แล้วใช้คำสั่ง attrib เพื่อแสดงไฟล์ที่ถูกซ่อนอยู่ใน USB drive โดยใช้คำสั่งดังนี้:
    • attrib -h -r -s /s /d ชื่อเส้นทางของ USB drive
    • attrib -h -r -s /s /d G:\*.*
  • 3. ลบไฟล์ที่ไม่ได้ต้อง: หากมีไฟล์ที่ไม่รู้จักหรือไม่ได้ต้องการบน USB drive คุณควรลบทิ้งเพื่อป้องกันการแพร่กระจายของไวรัส
  • 4. การคืนค่าระบบ: หากมีไฟล์ที่ไม่สามารถลบได้หรือมีปัญหาในการเข้าถึงข้อมูลบน USB drive คุณอาจต้องใช้คำสั่ง format เพื่อลบข้อมูลทั้งหมดและกำจัดไวรัส โปรดทราบว่าการฟอร์แม็ตจะลบข้อมูลทั้งหมดบน USB drive ดังนั้น แน่ใจว่าคุณมีการสำรองข้อมูลที่สำคัญก่อนดำเนินการ
  • 5. ป้องกันการติดเชื้อในอนาคต: เพื่อป้องกันการติดเชื้อจากไวรัสในอนาคต ควรใช้โปรแกรมตรวจจับไวรัสอย่างสม่ำเสมอ เปิดใช้งานฟีเจอร์การสแกนแบบ Real-time Protection (การป้องกันแบบเรียลไทม์) หากเป็นไปได้
โปรดทราบว่าการลบไวรัสอาจไม่สามารถทำได้ด้วยวิธีการธรรมดาและอาจต้องใช้เครื่องมือแบบพิเศษหรือช่วยจากผู้เชี่ยวชาญในบางกรณี อย่างไรก็ตาม การติดตั้งและประสานงานกับโปรแกรมตรวจจับไวรัสที่ดีสามารถช่วยลดความเสี่ยงของการติดเชื้อไวรัสในอนาคตได้มาก โปรดทำความเข้าใจและใช้ข้อมูลด้านความปลอดภัยอย่างระมัดระวังทุกครั้งที่ใช้งานอุปกรณ์ USB ของคุณ

สูตรทำผัดกะเพราหมูสับ

  • ส่วนผสม:
    • หมูสับ 200 กรัม
    • กระเทียมสับ 4-5 กลีบ
    • พริกขี้หนูสับ 5-10 เม็ด (ตามความชอบ)
    • ใบกะเพรา 2 กำมือ
    • น้ำมันพืชสำหรับผัด
  • สำหรับน้ำจิ้ม:
    • น้ำตาลทราย 1 ช้อนชา
    • น้ำปลา 1 ช้อนโต๊ะ
    • ซอสหอยนางรม 1 ช้อนโต๊ะ
  • วิธีทำ:
    • ตั้งกระทะหรือกระทะทองบนเตาแก๊ส ใส่น้ำมันพืชลงไป 1-2 ช้อนโต๊ะ แล้วรอให้น้ำมันร้อนอย่างใช่ได้
    • ใส่กระเทียมสับและพริกขี้หนูสับลงไปผัดจนหอม
    • เพิ่มหมูสับลงไปผัดพร้อมกับกลิ่นของกระเทียมและพริก
    • เมื่อหมูสุกแล้ว ใส่ใบกะเพราลงไปผัดจนสุก
    • ปรุงรสด้วยน้ำตาลทราย น้ำปลา และซอสหอยนางรม คลุกเคล้าให้เข้ากัน ปิดไฟ และตักใส่จานพร้อมกับข้าวสวย
